Pocketmatch Bot
===============

Описание
--------
Pocketmatch Bot — это Telegram-бот в стиле «тиндера». 
Он позволяет пользователям:
- создать и редактировать свой профиль,
- просматривать анкеты других людей,
- ставить лайки и получать «матчи»,
- получать уведомления,
- изменять параметры поиска и настройки.

Структура проекта
-----------------
- bot.py — основной файл запуска бота.
- config.py — конфигурация (токен бота, параметры БД).
- database.py — работа с базой данных.
- keyboards.py — клавиатуры и кнопки для интерфейса Telegram.
- requirements.txt — список зависимостей.
- start_bot.bat — удобный запуск на Windows.
- handlers/ — папка с обработчиками логики:
  * start.py — начало работы, приветствие.
  * profile.py — создание и редактирование профиля.
  * search.py — поиск пользователей.
  * matches.py — система лайков и совпадений.
  * notifications.py — отправка уведомлений.
  * settings.py — настройки пользователя.

Установка и запуск
------------------
1. Установите Python версии 3.9 или выше.
2. Скачайте и распакуйте архив с ботом в удобное место.
3. Перейдите в папку проекта (там где находится requirements.txt).
4. Установите зависимости:
   pip install -r requirements.txt
5. В файле config.py укажите:
   - TOKEN — токен вашего Telegram-бота (получается у @BotFather).
   - параметры базы данных (если используются).
6. Запуск бота:
   - через консоль: python bot.py
   - на Windows можно запустить start_bot.bat двойным кликом.

Использование
-------------
После запуска бот работает через Telegram.  
Пользователь может:
- зарегистрировать профиль,
- просматривать анкеты других пользователей,
- ставить лайки,
- получать уведомления о совпадениях,
- изменять свои настройки.
